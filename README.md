VortSerĉilo
===========

VortSerĉilo is an [Esperanto][]-to-English dictionary program.  Its
name literally means ‘word-searching tool’.  The program uses Paul
Denisowski’s [ESPDIC][] as its dictionary file.


Requirements
------------

* [Python 3][] or later.
